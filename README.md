# 欢迎使用flygrace组件库

## ✨ 特性

- 🌈 一个轻量级，优雅，便于使用的UI组件库。
- 📦 一组高质量的React组件。
- 🛡  用TypeScript编写，具有可预测的静态类型。
- ⚙️ 使用jest测试框架，为组件稳定性提供保障。

## 📦 安装

```bash
npm install flygrace
```

## 🔨 示例

```jsx
import { Button } from 'flygrace';

const App = () => (
  <>
    <Button btnType="primary">PRESS ME</Button>
  </>
);
```
